# ICU-Database-Management-System

## Overview
- Collaborated with a team of 3 students to design and implement an ICU Database Management System using Oracle SQL Developer, ensuring data integrity through primary keys, functional dependencies, and normalization with 3NF/BCNF algorithms.
- Developed and deployed a Unix Shell Menu and Web-UI that enabled users to efficiently Drop/Create/Populate/Query tables, strengthening teamwork and communication skills while improving database usability for medical data management.
